# Flextape.py

Flextape but it's python!

# Example/Usage

```python
import flextape
print(flextape.slap(["I sawed this ", "boat in half!"]))
print(flextape.slap(["I used ", str(10), " coats of Flex Seal on my air boat!"]))
```
